# Koverto - UUID

Wraps github.com/google/uuid for use as a protobuf type and with implementations
of various serialization interfaces.

## Usage

```protobuf
import "github.com/koverto/uuid@v1.1.0/uuid.proto";

message User {
    uuid.UUID id = 1;
}
```
